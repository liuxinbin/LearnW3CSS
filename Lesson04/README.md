## 知识点 

* w3-container

### w3-container

为了统一网页元素在各种浏览器以及各种终端上的显示，采用[w3-container]的元素来规范下列属性。

* margins
* paddings
* alignments
* fonts
* colors

同时[w3-container]可以用在下列html元素当中。

* div
* article
* section
* header
* footer
* form

## 综合例

~~~html
<div class="w3-light-gray">
    <p>任天堂主机switch要发卖了，它真的能取得成功吗？</p>
</div>
<div class="w3-container w3-light-gray">
    <p>任天堂主机switch要发卖了，它真的能取得成功吗？</p>
</div>
~~~
